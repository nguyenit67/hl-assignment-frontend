## Prerequisites

- Node.js and npm

## Deploy to Netlify

1. Install the Netlify CLI.
2. Create a new site using ntl init.
3. Deploy using ntl deploy.

```sh
# Install the Netlify CLI
$ npm install -g netlify-cli

# Create a new site in Netlify
$ ntl init

# Deploy to a unique preview URL
$ ntl deploy
```
